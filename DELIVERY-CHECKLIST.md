## 📧 고객 전달 방법

### 방법 1: GitHub 저장소 (추천)
1. Private 저장소 생성
2. 고객을 Collaborator로 초대
3. 설정 가이드 문서 제공

### 방법 2: 압축 파일 전달
1. 프로젝트 전체를 ZIP으로 압축
2. 이메일 또는 클라우드 스토리지로 전달
3. 압축 해제 후 설정 가이드 따라 진행

## 🎓 고객 교육 자료

### 기본 사용법
1. **색칠하기**: 지도에서 왼쪽 클릭
2. **색 지우기**: 지도에서 우른쪽 클릭  
3. **색상 변경**: 좌측 팔레트에서 색상 선택
4. **정보 저장**: 필지 클릭 → 정보 입력 → 저장
5. **백업하기**: 상단 백업 버튼 클릭

### 고급 기능
- 검색 ON/OFF 토글
- 색칠 모드 ON/OFF 토글
- 자동 백업 (매일 오전 2시)
- 수동 백업 및 복원
