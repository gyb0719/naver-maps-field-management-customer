🔄 자동 백업

시스템이 매일 새벽 2시에 자동으로 백업을 수행합니다. 최근 30일간의 백업이 보관되며, 오래된 백업은 자동으로 삭제됩니다.

📁 수동 백업

언제든지 '수동 백업 실행' 버튼을 클릭하여 즉시 백업을 생성할 수 있습니다. 백업 파일은 컴퓨터에 자동으로 다운로드됩니다.

🔄 데이터 복원

백업 파일을 업로드하여 데이터를 복원할 수 있습니다. 복원 전 현재 데이터는 자동으로 안전 백업됩니다.

⚠️ 주의사항

데이터 복원 시 기존 데이터가 덮어쓰여집니다. 중요한 작업 전에는 반드시 수동 백업을 수행하세요.
